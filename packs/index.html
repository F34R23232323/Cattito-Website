Packs are guaranteed sources of cats. This is the **internal mechanic** detailing how the pack rarity is determined and how many cats you receive upon opening.

Opening a Pack: Step-by-Step

  1. A pack starts with some **specific rarity** (e.g., a "Wooden Pack" starts at Wooden rarity).
  2. **Rarity Upgrade Chance:** The pack's rarity is determined through a loop:
    • **70% chance:** The current rarity becomes the **final pack rarity**. Proceed to Step 3.
    • **30% chance:** The pack is **upgraded by one rarity** and Step 2 is repeated. (Note: Celestial is the maximum rarity and cannot be upgraded further).
  3. Once the final rarity is chosen, get its **Base Value** from the table below:
    Pack Rarity Base Value
    Wooden65
    Stone90
    Bronze100
    Silver115
    Gold230
    Platinum630
    Diamond860
    Celestial2000
  4. A random **Cat Type** is picked. Every cat type has the same chance of being chosen: **1/22**.
  5. The result number of cats to reward is calculated: Pack Base Value / Cat Type Value.

  6. The **whole part** of the result number is the guaranteed reward.
  7. The **fractional part** represents the chance of getting one *additional* cat.
    **Example:** If the result is 42.69, you are guaranteed **42 cats**. You will receive an extra cat (**43 total**) in **69% of cases**, and **42** in the remaining 31%.
  8. **Fallback:** If the result reward is nothing (i.e., less than 1), the reward falls back to **1 Fine cat**.
