Cattlepass

The Cattlepass System

The Cattlepass is accessed by running /battlepass. Seasons reset **monthly** (1st 00:00 UTC), each with a different set of rewards. Each level requires XP to **level up and get the reward**. XP is obtained by **completing quests**, which show in the Cattlepass menu. Quest XP ranges from minimum to maximum, always multiples of 10.

XP per level is likely global across all seasons. XP increases by 50 every three levels (e.g., Levels 1-3 require **550**, Levels 4-6 require **600**). Bonus levels (31+) require **1500 XP** each.

The Packs Update

Season 3 added **Cat Packs** acquired from Cattlepass levels. 8 tiers: Wooden, Stone, Bronze, Silver, Gold, Platinum, Diamond, Celestial. Each pack has a 30% chance to upgrade (except Celestial) and gives a random cat type. See Cat Packs for details.
