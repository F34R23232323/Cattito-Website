Custom Cats

These **exclusive cats** can only be obtained through the official Discord server (Cat Stand) and by trusted members. Any cats outside of the above table are Custom cats, placed after the eGirl cat in inventory. They have **no value** and only one type can be had at a time. Custom cats cannot be /trade-d or /gift-ed.

The amount of the custom cat can be changed using cat!amount (amount). Images of some Custom cats can be found here.
