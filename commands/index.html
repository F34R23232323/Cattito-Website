Cattito Commands Wiki

NOTE: Command arguments in square brackets [] are optional, in parentheses () are required. Defaults are shown as **[argument=default]**.

Setup Commands

/changeavatar

Changes Cattito's profile picture.

/changemessage

Changes the cat spawn or cat catch message.

/changetimings [minimum_time=120] [maximum_time=1200]

Changes the timings for a channel. Requires minimum_time $\ge 20$ and $\le$ maximum_time.

/forget

Makes Cattito stop spawning cats in the desired channel.

/help

Learn how to use the bot, setup, and play. Contains info about achievements.

/setup

Makes Cattito start spawning cats in the desired channel. If permissions are missing, it will list them.

/wiki

Sends quick links to Cattito wiki pages:

  • Cattito
  • Cat Spawning
  • Commands
  • Cat Types
  • Cattlepass
  • Achievements
  • Packs
  • Trading
  • Gambling
  • The Dark Market
  • Prisms

Most Used Commands

/achievements

Shows your achievements and unlocks Dark Market access.

/battlepass

Monthly progression system for quests and XP. Tracks level and quest reminders.

/catalogue

Shows value, spawn chance, and number of each cat type discovered.

/gift (user) (type) [amount=1]

Gift cats, packs, or rain. Tax applies for $\gt 5$ units. Grants achievements and quest progress.

/inventory [user=yourself]

Displays all cats, stats, achievements, and Cattlepass level. Inventory can be edited.

/last

Shows last cat caught, next spawn estimate, and remaining rain cats.

/leaderboards [type=Cats]

Top 15 users for a type/category: total cats, value, fastest/slowest, battlepass XP, cookie count, pig score.

/packs

Info on Cat Packs, and open them.

/prism

Info on prisms, craft and configure them.

/stats [user]

List stats of a user.

/trade (user)

Trade cats, packs, prisms, and rain. Grants achievements and quest progress. See Trading.

Supporting Commands

/bless [SUPPORTER-ONLY]

Bless random catches with double cats, stacking with Cataine. Optional anonymity. Very low chance per total rain minutes.

cat!amount (amount) [SUPPORTER-ONLY]

Change the amount of a custom cat. Regular limits apply.

/editprofile [color=staring] [emoji=none] [image=none] [SUPPORTER-ONLY]

Customize inventory appearance.

/rain

Get info about Cat Rains and start them if you have rain minutes.

/store

Link to buy Cat Rain minutes: Store

/vote

Vote every 12 hours for quests in Cattlepass. Voting every 24 hours increases streak.

Random Commands

/8ball

Replicates the real 8-ball with added negative responses. Grants achievements and quest progress.

/bal

Sends an image of cat coins.

/brew

Bot refuses to brew coffee. Grants achievement.

/cat [type=Staring]

Sends a cat image of the specified type.

/cursed

Sends a cursed image.

/daily

Replies with a humorous "no daily cats" message. Grants achievement.

/define (word)

Find definition from AH Dictionary or Wordnik.

/fact

Gets a random cat fact. Grants achievements.

/fake

Sends a fake eGirl cat message. Grants achievement.

/getid (thing)

Gets the user ID of a person or item.

/ping

Shows ping of bot and server shard. Completes a quest.

/rate (thing) (stat)

Sends "thing is X% stat" with special rules.

/random

Gets a random cat image. Grants achievement.

/remind [days=0] [hours=0] [minutes=0] [text=Reminder!]

Sets a reminder via DM. Grants achievement and quest progress.

/roll [sides=6]

Roll a dice from 1 to number of sides. Special cases for 1, 2, 0, or negative sides.

/tiktok (text)

Reads text using TikTok TTS. Grants achievement and quest progress.

Minigames

/casino

Gambling minigame. Impossible to win. Grants achievement.

/cookie

Cookie clicker simplified game. Grants achievements.

/roulette

Bet cat dollars on red, black, green, or number.

/rps [person=EVERYONE]

Rock-Paper-Scissors with someone or everyone. Cattito always wins.

/slots

Slot machine minigame, no risk, can remove negative cats.

/tictactoe (person)

Start Tic-Tac-Toe. Grants achievement if you win. Cattito AI included.

/pig

Starts Pig game with best score. Grants achievement.

Miscellaneous

/credits

Shows all contributors, APIs, and open source projects.

/info

Shows system specs, status, and stats about the bot.

car

Other Admin Commands

/giveachievement (user) (name)

Give any achievement to a user (with exceptions).

/givecat (user) (amount) (cat-type)

Give cats to a user.

/forcespawn [type=random]

Spawns a cat automatically with optional type. Private error messages if invalid.

/preventcatch (person) (timeout)

Prevents a person from catching cats for a period.

/nuke

Resets all data in the current server.

/reset (user)

Clears user data.

Developer Commands

cat!custom (user) (name)

Gives a custom cat.

cat!eval (code)

Executes code. Shows exception if occurs.

cat!news (announcement)

Makes Cattito send announcement in all /setup-ed channels.

cat!print (text)

Bot says text in channel.

cat!rain (user) (minutes)

Gives cat rain to a user for minutes.

cat!restart

Restarts bot after all cat rains conclude.

cat!sweep

Resets channel spawn status.
